Projects

Distributed Key-Value Store

Developed an in memory key-value store (Distributed Systems course project) with features like Detection of Strict Consistency or Eventually consistent and Support for Consistency levels. Last fall I was a software engineering intern at Facebook in Menlo Park. There, I worked on the Support Interfaces team under the Protect and Care org. I implemented a new Support Inbox item type that notifies victims of tag spam that the content they were tagged in was taken down by Facebook for spam. With this new feature, victims are now aware of the efforts that Facebook takes to make the platform more spam-free. By informing more people that action was taken against spam, this will hopefully encourage more spam reporting behavior. This was done with a little help from Hack (Facebook's PHP language), Haskell, SQL, and Thrift.


Manhattan Nest

Developed a map visualization tool for housing recommendation. The recommendation is based on factors like cost of living, crime rate, night life, proximity to subways, hospitals and restaurants; using Yelp API and Open data source.


CitiBike

Developed an analytics application aimed at categorizing customers to facilitate targeted advertising, which will help to redesign marketing strategies and to improve customer base. Features: Google estimate analysis, bike rebalancing, boroughs distribution, effects of subways and seasons.


Neighbor Connect: Facebook for Neighbors

Social platform for neighbors and friends to connect. Users are being allocated a specific block and neighborhood based on their addresses. Designed RESTful APIs using flask micro framework and SPA using AngularJS. Features: Users can post messages and comments, add/remove friends, access data feed from the neighborhood/block/friends group and send private messages.

Privacy Protection on Transfer system for ATM from Brute force attack

Developed a two way Authentication system by generating QR code and embedded One Time Password. The webcam in Automatic Teller Machine is also triggered to capture the illegal access.

As the ACM secretary, I was also responsible for keeping meeting minutes, updating the ACM officer website, delegating tasks among all the ACM officers, and making sure that we were progressing as planned. For more information on UW ACM, please check out http://www.cs.washington.edu/acm/.
